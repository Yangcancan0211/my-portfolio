# 🚀 前端逻辑实战作品集 (JS & Ajax Projects)

## 📁 项目速览

### 1\. 🌤️ 实时天气预报

  - **手写 `myAxios`**：不依赖第三方库，基于 `Promise` 和 `XMLHttpReqeust` 手写封装了 Ajax 工具。
  - **动态搜索**：城市搜索联想建议，点击自动切换全屏天气状态。
  - **多日预报**：解析复杂的嵌套 JSON，一次性渲染未来一周的天气。

### 2\. 📚 图书管理系统

  - **完整业务**：实现了图书的**增加、删除、修改、查询**完整闭环。
  - **高效表单**：使用插件快速序列化表单数据，并处理了编辑状态下的**数据回显**。
  - **弹窗交互**：整合 Bootstrap 模态框，确保操作反馈流畅。

### 3\. 🛍️ 华为商城：轮播图组件

*关键点：动画性能、定时器管理*

  - **无缝轮播**：利用 DOM 克隆技术实现无限循环滚动。
  - **性能优化**：监听 `onvisibilitychange`，当用户离开网页时自动停止动画，节省内存。

### 4\. 📦 华为商城：商品列表

*关键点：数据驱动渲染、Flex布局*

  - **自动化渲染**：循环解析 JSON 商品数据，动态创建商品卡片。
  - **布局补位**：专门处理了 Flex 布局最后一行不满时的对齐问题。

-----

## 🛠️ 技术栈

  - **核心**：JavaScript (ES6+)、DOM 操作、异步编程 (Promise/Ajax)。
  - **样式**：CSS3 (Flexbox/Grid)、响应式设计、CSS 模块化拆分。
  - **工具**：Axios、Bootstrap 5。
